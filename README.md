## Descripción del proyecto
Este proyecto consiste en el desarrollo de una escena de Realidad Virtual (VR) utilizando Unity y XR Interaction Toolkit, enfocada en la implementación de interacciones básicas aprendidas en clase.  
La aplicación permite al usuario interactuar con objetos dentro de un entorno virtual mediante el uso de manos virtuales, realizando acciones como agarrar, soltar y detectar colisiones, simulando una experiencia inmersiva en VR.

El proyecto fue desarrollado siguiendo los procedimientos vistos en clase y adaptado de manera individual.

## Funcionalidades implementadas
- Configuración de un proyecto VR en Unity.
- Integración de XR Interaction Toolkit.
- Configuración de XR Origin (XR Rig).
- Uso de controladores basados en acciones (Action Based Controllers).
- Implementación de manos virtuales animadas mediante el paquete **Oculus Hands**.
- Interacción de objetos que se pueden:
  - Cambio de color al tocar
  - Agarrar
  - Soltar
  - Detectar colisiones físicas
- Escenario con plano (suelo) y mesa para interacción con los objetos.

## Instrucciones para ejecutar el proyecto
1. Abrir **Unity Hub**.
2. Seleccionar la versión de Unity utilizada en el proyecto.
3. Abrir la carpeta del proyecto desde Unity Hub.
4. Verificar que el paquete **XR Interaction Toolkit** esté instalado.
5. Asegurarse de que el **Input System** esté configurado correctamente.
6. Ejecutar la escena principal desde Unity.
7. (Opcional) Conectar un visor Oculus Quest mediante Meta Quest Link para pruebas en VR.
